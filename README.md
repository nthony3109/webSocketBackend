# WebSocket Backend System

This is a backend project built to understand how WebSocket communication works using Spring Boot.

The project demonstrates the basic setup and configuration of WebSocket in a Spring application and how messages can be sent and received in real time.

## Features

- Spring WebSocket configuration
- Real-time message communication
- Basic WebSocket endpoint setup
- Usage of `SimpMessagingTemplate` for formatting and sending messages
- Lightweight project with no database connection
- Simple architecture for learning purposes

## Technologies Used

- Java
- Spring Boot
- Spring WebSocket
- Maven

## Purpose of the Project

The main goal of this project is to learn and understand:

- How WebSocket works in backend systems
- Real-time communication between client and server
- Message broadcasting using Spring Boot
- WebSocket configuration and message handling
